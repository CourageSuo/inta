PhantomJS课程概述

PhantomJS是一个此时无形胜有形的浏览器，它可以通过javascript的接口操作网页中的DOM，JSON，Canvas等

第一节：安装
Windows安装：
下载解压文件
我的电脑 -> 属性 -> 高级系统设置
点击环境变量
在系统变量窗口中，找到Path变量，选中然后选择编辑
新建后，添加新的环境变量（phantomjs.exe）所在路径
在cmd中，运行phantomjs
Mac安装：
下载解压文件
在用户目录下查看ls -a查看全部文件
打开.bash_profile文件
加入export PATH=$PATH:/Users/suoyong/phantomjs-2.1.1-macosx/bin
在终端中输入phantomjs运行

第三节：WebPage对象（一）
创建WebPage对象：create()
打开网址：page.open
在evaluate方法中操作页面（DOM，JSON，Canvas等）
把当前页面保存成图片:page.render()
第四节：WebPage对象（二）
沙盒：在一个固定的地方有限制的执行程序，保证不会干扰系统中其他程序运行.
evaluate方法在沙盒中运行，沙盒中的输出不等于javascript中的输出
在沙盒中使用console.log()需添加page.onConsoleMessage = function(msg) { console.log(msg)}
可为evaluate添加参数，并在回调函数中使用
第五节：提交表单
page.render()可以把页面以图像的模式保存在电脑中
page.onLoadFinishied = function(){ }
1）直接使用dom对象form.submit()
2) 使用事件触发，var evt = docuemnt.createEvent('MouseEvents')
evt.initMouseEvent('click')
dom.dispatchEvent(evt)
第六节：操作Cookie
phantom.cookies:查看cookie
phantom.addCookie:设置cookie，（必须包括：name,value,domain键）
page.cookies:查看cookie
page.addCookie:设置cookie, (必须包括：name,value,domain键)
第七节：CasperJS
CasperJS是phantomjs的扩展，能更加简单的操作页面元素
casper.start() = page.open()
echo() = console.log()
下载CasperJS学习文件
第八节：步进式脚本语言
CasperJS属于步进式脚本语言（Navigation Scripting)
使用casper.then()添加执行步骤
在casper.run()执行每一步的任我
casper.page = webpage
casper.capture = webpage.render
casper.evaluate = webpage.evaluate
